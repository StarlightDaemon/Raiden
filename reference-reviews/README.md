# Reference Reviews

This directory holds the review output for imported prototype repos.

## Standard Layout

For each imported repo, create:

- `reference-reviews/<repo-name>/REPO_TOOLING_REVIEW.md`
- `reference-reviews/<repo-name>/IMPORT_CANDIDATES.md`

Use the templates in `reference-reviews/templates/`.

## Cross-Repo Comparison

Use `CROSS_REPO_MATRIX.md` to compare prototype repos side by side before promoting anything into RAIDEN canon.
