# Stargate Reference Extracts

## Purpose

This folder contains preserved high-value patterns extracted from the `Stargate` legacy holding snapshot.

The raw snapshot was retired to reduce repo noise, but its lightweight project-registry and stewardship/maturity-scoring patterns remain relevant for RAIDEN's sprawl-control and handoff discipline.

## Contents

- `registry-and-stewardship-pattern.md`: Contains the prototype-sprawl control, maturity category scoring, and registry JSON schema patterns.

## Provenance

These patterns were extracted from `reference-repos/Stargate/FINAL_STEWARDSHIP_REPORT.md` and `reference-repos/Stargate/stargate-registry.json`. They are preserved here for potential future reuse in RAIDEN's toolkit/governance canon.
